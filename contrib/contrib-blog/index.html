Add Tutorials

This section regroups the guidelines that should be followed to write a tutorial for the ArchLan Linux documentation.

Prerequisites‚Äã

  • To work locally, setup a local development environment on your machine.
  • Ensure to have basic understandings of the Markdown and MDX syntaxes. See Docusaurus Markdown Features for further information.
  • Make sure that you have the ability to fork, clone a repository and create a pull request.

Format structure and file organization‚Äã

All the .md and .mdx files for tutorials are contained within the blog directory.

  • Simply create a new file with the appropriate format to add a tutorial to the wiki.

Add the correct slug, tags and title in the heading of the file you want to add.

title: TITLE
authors: ID
tags: [TAG1, TAG2, TAG3]
---
