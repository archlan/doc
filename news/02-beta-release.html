An improved beta v0.2 is now out!

January 17, 2022
Lanhild
Developer/Maintainer of ArchLan Linux

Newer and better! To check it out or install it on your machine, head to ArchLan:Download.

Even though there is still place for improvement, this release is much better and much more usable.

Read this post for more information about this release:

note

Keep in mind that this release is a beta and I am actively developing ArchLan, which means that you might encounter bugs or unexpected errors while using it. The next versions of ArchLan will be even better and I will make sure to listen to users suggestion in order to make ArchLan better!

Verify the authenticity of the ISO‚Äã

  • Download the ISO, GPG signature and the sha256sum files
  • Ensure that all the files you have downloaded are present in the same directory as the ISO
  • Verify the key on the keyserver:
$ gpg --recv-keys 5F765E0E531F5B53
  • Verify the iso signature:
$ gpg --verify archlan-2022.01.16-x86_64.iso.sig
  • Then verify the sha256sum:
$ sha256sum -c archlan-2022.01.16-x86_64.iso.sha256sum
