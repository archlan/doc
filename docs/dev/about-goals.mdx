# About ArchLan
>This section regroups my learning goals with ArchLan, and also objectives or
>requirements I want to meet at each new release.

## Learning goals
- Make sure to never have more than **800 packages** on the final installation

- Make the most out of the [official package repo](https://github.com/archlan/archlan-packages)

- Learn how to use/create a GPG key for package signing

- Learn how to optimize the speed of a Linux system by minimizing the number of packages and files

- Improve my knowledge of the process of creating and maintaining a distribution

- Exclusively use window managers

- Have an ISO file smaller than 2.0GB
